# 好多店 AI 岗位 / AI 员工 / Skill / 流程 / Case 关系定义 V0.1

## 文档目标

定义好多店 AI Native 业务执行系统中的核心对象：

-   AI岗位（Role）
-   AI员工（Employee）
-   Skill
-   Workflow
-   Case
-   Tool
-   Run

目标是在好多店组织中创建 AI 业务岗位，让 AI
像企业员工一样承担明确职责，通过流程、技能和工具完成业务闭环。

------------------------------------------------------------------------

## AI岗位（AI Role）

AI岗位是业务职责的抽象，定义：

-   角色使命
-   工作职责
-   能力范围
-   权限边界
-   禁止事项

示例：

AI风险督导：

负责风险分析、责任定位、整改推动和闭环跟进。

禁止：

-   修改历史业务数据
-   绕过审批
-   自动处罚

------------------------------------------------------------------------

## AI员工（AI Employee）

AI员工是 AI岗位 的实例。

关系：

AI岗位 → 实例化 → AI员工

例如：

AI风险督导岗位：

-   华东区域 AI风险督导
-   华南区域 AI风险督导

不同员工具有：

-   不同负责范围
-   不同权限
-   不同数据范围

------------------------------------------------------------------------

## Skill（技能）

Skill 是 AI员工完成业务任务的方法能力。

AI风险督导 Skill：

### 风险分析 Skill

输入：

-   风险事件
-   门店信息
-   检查项
-   历史记录

输出：

-   风险原因
-   证据
-   影响分析

### 整改方案 Skill

输出：

-   整改措施
-   责任人
-   时间要求
-   复查方式

### Case跟进 Skill

负责：

-   是否完成
-   是否延期
-   是否升级

------------------------------------------------------------------------

## Workflow（流程）

Workflow 描述：

什么时候启动 AI员工，由哪个 AI员工，在什么步骤调用什么 Skill。

示例：

风险整改流程：

风险规则触发

↓

AI风险督导

↓

风险分析Skill

↓

责任定位Skill

↓

整改方案Skill

↓

人工审批

↓

创建整改任务

↓

Case持续跟进

------------------------------------------------------------------------

## Case（业务案例）

Case 是 AI员工持续跟进的业务对象。

示例：

门店A冰机连续3次不合格。

创建：

门店A冰机整改事件 Case

包含：

-   风险记录
-   AI分析结果
-   整改任务
-   责任人员
-   审批记录
-   跟进记录
-   最终结果

------------------------------------------------------------------------

# AI岗位规划

## AI风险督导（第一期）

定位：

区域督导助手。

职责：

-   接收风险事件
-   分析问题
-   定位责任
-   生成整改方案
-   推动闭环

数据：

-   巡店结果
-   检查项
-   风险规则
-   门店信息
-   人员关系
-   整改记录

## AI店长助手（第二阶段）

定位：

店长执行助手。

职责：

-   解释问题
-   提醒任务
-   生成工作计划
-   辅助整改

## AI运营经理（第二阶段）

定位：

总部运营助手。

职责：

-   区域分析
-   经营报告
-   趋势发现
-   决策辅助

------------------------------------------------------------------------

## 第一阶段范围

只实现：

AI风险督导。

覆盖：

-   巡店检查结果
-   风险规则
-   门店信息
-   人员关系
-   整改任务
-   工单闭环

暂不接入：

-   视频AI
-   客流
-   财务
-   业绩分析
